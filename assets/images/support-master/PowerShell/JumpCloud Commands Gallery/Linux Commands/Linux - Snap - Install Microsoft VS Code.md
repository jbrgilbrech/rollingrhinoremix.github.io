#### Name

Linux - Snap - Install Microsoft VS Code | v1.0 JCCG

#### commandType

linux

#### Command

```
snap install code --classic
```

#### Description

Install Microsoft VS Code app with the Snap app manager. To learn more about Snap, visit [SnapCraft's website](https://snapcraft.io/)
